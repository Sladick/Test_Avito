Необходимо распарсить любую страницу на авито и получит из объявлений максимально возможное количество данных.

Получить с сайта ЦБ актуальный кур евро и добавить в полученную с Авито таблицу цену на товар в Евро.

Полученные данные записать в базу данных Sqlite и дополнительно вывести в виде отчета в Exel.
